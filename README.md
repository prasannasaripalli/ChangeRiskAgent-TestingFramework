This project is a beginner-friendly DeepEval validation layer to a Change Risk AI Agent.

It includes:

- Existing Streamlit app: `app.py`
- Existing agent logic: `agent_utils.py`
- FastAPI wrapper for black-box API tests: `api.py`
- DeepEval `Golden` dataset
- `EvaluationDataset.evals_iterator()` based tests
- LLM metrics
- RAG metrics
- Component-level tracing around retriever, risk scoring, and LLM analysis
- Confident AI observability support
